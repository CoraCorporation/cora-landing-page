# CORA Landing Page

Página de aterrizaje oficial de **CORA**, un sistema inteligente de procesamiento de documentos que permite convertir archivos PDF de gran tamaño en documentos Word estructurados y editables.

Esta landing ha sido diseñada como una interfaz ligera, moderna y orientada a la presentación del producto, desplegada mediante GitHub Pages.

---

## 🚀 Descripción del proyecto

CORA es una solución enfocada en la automatización del procesamiento documental, especialmente en escenarios donde se requiere transformar archivos PDF complejos en formatos editables sin perder estructura.

Esta landing page representa la capa de presentación del sistema, con un diseño centrado en:

- Claridad en la propuesta de valor  
- Experiencia de usuario limpia  
- Rendimiento óptimo en carga  
- Estructura escalable para futuras integraciones
